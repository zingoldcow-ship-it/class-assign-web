# 반배정 웹앱 v3 (브라우저 전용)

- GitHub Pages 같은 정적 호스팅에서 URL로 바로 사용합니다.
- 학생 엑셀(.xlsx)은 서버로 전송되지 않고, 브라우저 안에서만 처리됩니다.
- v3 기능: 반배정 실행 + 가중치 + 분리/배려 코드(강도) + 결과 새 탭 + 결과 엑셀 다운로드

## 파일
- index.html : 설정/업로드/실행 화면
- result.html : 결과 화면(새 탭)
- app.js : 반배정 계산 로직
- result.js : 결과 시각화/다운로드

## 권장 엑셀 열
학생명, 성별, 학업성취, 교우관계, 학부모민원, 특수여부, ADHD여부, 분리요청코드, 배려요청코드, 비고
