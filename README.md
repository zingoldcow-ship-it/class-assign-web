# 반배정 웹앱 (브라우저에서만 실행) v2

✅ 목표: URL만으로 접속 가능한 웹앱 뼈대 만들기  
✅ 개인정보: 학생 엑셀은 서버로 업로드하지 않고, 브라우저에서만 읽습니다.

## 실행(로컬 테스트)
- 이 폴더를 VSCode에서 열고 Live Server로 열거나,
- 간단히는 파일 더블클릭으로도 열리지만(일부 브라우저에서 제한될 수 있음) Live Server 권장.

## 배포(GitHub Pages)
1) 이 폴더를 GitHub 저장소에 업로드
2) Settings → Pages → Source: Deploy from a branch
3) Branch: main / root 선택 → Save
4) 몇 분 후 Pages URL 생성
